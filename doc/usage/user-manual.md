## Getting Help

If you need help while using **ticat**, you can:

- Use the `+` command for detailed help: `ticat :+`
- Use the `-` command for brief help: `ticat :-`
- Check the [troubleshooting guide](../TROUBLESHOOTING.md) for common issues
- Open an issue on [GitHub](https://github.com/innerr/ticat/issues)

## Learning Path

### For New Users
1. Start with the [Quick start](../../README.md) to understand the basic concepts
2. Learn [Basic operations](./basic.md) to build and run commands
3. Explore [Hub management](./hub.md) to get modules from the community

### For Advanced Users
1. Deep dive into [Environment management](./env.md) for configuration control
2. Master [Flows](./flow.md) for workflow automation
3. Use [Command searching](./cmds.md) to discover available tools

### For Module Developers
After mastering the basics, check out the [Module development guide](../quick-start-mod.md) to create your own modules.
