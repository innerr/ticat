# [Spec] Help

## The helping commands (commands provide info)
```
## Sequence info
$> ticat <command> : <command> : desc

## Hub info
$> ticat hub.list

## Flow info
$> ticat flow.list

## Command info
$> ticat cmds.list
$> ticat cmds.tree
$> ticat cmds.tree.simple

## Env info
$> ticat env.list
$> ticat env.tree

## Abbrs info
$> ticat env.abbrs

## Find command, env
```
